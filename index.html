1 Pricing varies based off of the quantity of employees and billing frequency

Features

The most unique thing about the website is that they have Navbars clearly listing every task and utility the user is capable of using on the website. Features like adding shifts and search buttons are made very obvious and easy to use. You can change the calendar grid display, the listing of the calendar itself, and which position is being viewed all from the main page after having logged in. The Navbar offers incredibly easy to find tools for things like editing the employee rosters, scheduling time off, and messaging (to name a few).

Features

The main page instantly shows the viewer unassigned shifts with avaiable shifts directly beneath it. There is an option to add graphs and statistics to the main page. Users can toggle between different schedule views easily. Users can also check assigned tasks, messages, their newsfeed, reports, and time clock.
